## 🚀 Instalasi

1. **Clone repository ini**
   ```bash
   git clone https://github.com/nizanularifin-cmd/Adimizanul.git
   cd Adimizanul
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```
   atau jika menggunakan yarn:
   ```bash
   yarn install
   ```

3. **Konfigurasi (jika diperlukan)**
   ```bash
   cp .env.example .env
   # Edit .env sesuai kebutuhan Anda
   ```
